# AZIKタイピング養成妖精 (azik-fairy) — ガイド

## 前提条件

本プロジェクトをローカルで起動・開発・テストするには、以下の環境が必要です。

- **Node.js**: `v20` 以上を推奨
- **パッケージマネージャー**: `npm`

---

## クイックスタート（ローカル開発手順）

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下を実行します。
```bash
npm install
```

### 2. 開発サーバーの起動
ローカル開発用のホットリロードサーバーを起動します。
```bash
npm run dev
```
起動後、ブラウザで `http://localhost:3000` にアクセスして動作を確認してください。

### 3. 単体テストの実行（TDD用）
Vitest によるテストスイートを実行します。
```bash
npm run test
```

### 4. 静的エクスポートビルド
静的ファイルを `./out` ディレクトリに出力します。
```bash
npm run build
```

## 基本的な遊び方と設定

### 1. ステージを選択して開始
- タイトル画面から **「GAME START」** を選択します。
- レベルタブ（Lev1 〜 Lev4、実戦練習）を切り替え、練習したいステージを選択します。
- プレイが始まると、問題の漢字とその下に「ひらがな（セグメント）」、さらにその下に「次に打つべきキー」が表示されます。

### 2. 設定のカスタマイズ（OPTIONS）
タイトル画面の **「OPTIONS」** から、以下の設定が可能です。

- **INPUT METHOD（入力モード）**
  - **STRICT（強制モード / 推奨）**: AZIKの短縮配置と異なる入力（通常のローマ字等）を完全に無視（ブロック）します。ミス数には加算されず、画面が揺れる警告が発生します。
  - **NORMAL（ノーマルモード）**: 通常のローマ字入力も受け付けますが、押し間違いはミス数に加算されます。
- **KEY GUIDE（キーガイド）**
  - 文字の下に次に押すアルファベット（例: `kz`）を表示するかどうか。
- **AZIK HELP TABLE（ヒントヘルパー）**
  - 画面下部に、現在入力中の文字列に対応するAZIKの解説ルールを動的に表示します。

## カスタムキーのインポート手順

アビリティを最大限に活かすため、自分が普段利用しているローマ字変換テーブル（TSV/SKK）から設定をインポートできます。

### A. Google日本語入力（TSV）からのインポート
1. Google日本語入力のプロパティ ➔ 「一般」タブ ➔ 「ローマ字テーブル設定」の「編集...」を開きます。
2. 「編集」メニューから「エクスポート...」を選択し、設定ファイル（TSV）を保存します。
3. 保存したテキストファイルを開き、中身をすべてコピーします。
4. 本アプリの **OPTIONS** ➔ **CUSTOMIZE KEYS** ➔ **「TSV/SKKインポート」** タブを開きます。
5. テキストエリアにコピーした内容を貼り付け、**「インポート適用」** ボタンをクリックします。

### B. SKK (`kana-rule.conf`) からのインポート
1. macSKKやAquaSKK等で利用している `kana-rule.conf`（または `.rule`）ファイルを開きます。
2. ファイルの内容（カンマ区切りのテキスト）をコピーします。
3. 本アプリの **OPTIONS** ➔ **CUSTOMIZE KEYS** ➔ **「TSV/SKKインポート」** タブを開きます。
4. コピーした内容を貼り付け、**「インポート適用」** をクリックします。

## トラブルシューティング

| 事象 | 原因 | 解決策 |
|------|------|--------|
| キーを押しても文字が進まない | 入力モードが `STRICT` になっており、通常のローマ字（例: `kan`）で入力しようとしている。 | 画面下部のヒント表やガイドを確認し、AZIKの短縮キー（例: `kz`）を押してください。または設定で `NORMAL` モードに切り替えてください。 |
| ページをリロードするとカスタムキーや星（進捗）が消えてしまう | ブラウザの `localStorage` が無効になっている、またはプライベート（シークレット）ブラウズモードで起動している。 | 通常ブラウザモードで起動するか、ブラウザ設定で `azik-fairy` のローカルストレージ使用を許可してください。 |
| 入力ガイドやヒントが表示されない | OPTIONSで各ガイドが `HIDE`（非表示）に設定されている。 | OPTIONS画面で `KEY GUIDE` および `AZIK HELP TABLE` を `SHOW` に設定してください。 |
