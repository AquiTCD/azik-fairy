# AZIKタイピング養成妖精 (azik-fairy) — 概要

## これは何？

**AZIKタイピング養成妖精 (azik-fairy)** は、日本語の入力速度を飛躍的に高める拡張ローマ字入力方式「AZIK」を、楽しくかつ強制的に習得するためのWebタイピングゲームです。
レトロポップなファミコン風（NES）のドット絵（ピクセルアート）デザインと、ユーザーの練習をサポートするギャル妖精のキャラクター（Fairy Helper）が特徴です。

## なぜ存在するのか？

AZIK（エイジック）は、打鍵数を劇的に減らすことができる非常に効率的な日本語入力方式ですが、独自の短縮マッピング（例：「かん」➔ `kz`、「せい」➔ `ss` 等）が多数存在するため、指と脳が馴染むまでに相応の訓練が必要です。
既存のタイピングソフトの多くは通常のローマ字入力のみを許容するため、AZIKの練習中につい慣れた通常ローマ字で打ってしまい、短縮キーが定着しにくいという課題がありました。

本アプリは、この習得プロセスにおける以下の課題を解決するために開発されました：
- **通常のローマ字入力を完全に無効化する強制モード**の提供。
- ローマ字マッピングルールごとに分類された**段階的なレッスン（Lev1〜Lev4）**。
- 日常会話、ビジネスメール、IT用語などの実用シーンに即した**実戦（Practice）ステージ**。
- Google日本語入力やSKKの定義ファイルをそのままコピペして利用できる**カスタムキーのインポート機能**。

### 主要なユースケース

- **AZIKの基礎から応用までを段階的に学ぶ**  
  「っ」や「ん」の単打（Lev1）から、撥音・二重母音の拡張（Lev2）、互換キー（Lev3）、特殊な文末ショートカット（Lev4）までをレッスン形式で学べます。各レッスンの最後には、学習成果を確認する文章テストが統合されています。
- **実戦シーンを想定したタイピング練習**  
  「プログラミング」「データベース」といったIT用語、および「いつもお世話になっております」といったビジネスシーンの文章練習を通して、実戦的な打鍵力を養います。
- **自己のキーマップに合わせた練習**  
  ユーザー独自のAZIKカスタムマッピングや、他デバイスで使い慣れたローマ字テーブルを読み込み、カスタムされたルール上でタイピングの判定・ガイド・ヒント表示を受けられます。
- **練習習慣と成長のトラッキング**  
  日々の練習による「累計打鍵数」や「連続ログイン日数（ストリーク）」、各ステージの「クリアスター数（★1〜★3）」をブラウザに自動記録し、成長を可視化します。

## 主要コンセプト

| 用語 | 定義 |
|------|---------|
| **AZIK** | QWERTY配列でのローマ字入力をベースに、子音と特定の母音拡張キーの組み合わせで日本語の打鍵数を削減する方式。 |
| **撥音拡張** | 「〜あん」「〜いん」などの撥音（〜ん）を、子音キーの直後に母音キーの1行下のキー（z, k, j, d, l）を押すことで2打鍵で入力する仕組み。 |
| **二重母音短縮** | 「〜あい」「〜うう」などの二重母音を、子音キーの直後に母音キーの隣のキー（q, h, w, p）を押すことで2打鍵で入力する仕組み。 |
| **強制スルーモード (STRICT)** | AZIKで定義された推奨打鍵（例: `kz`）以外の入力（例: `kan`）を受け付けず、キー入力を無視（スルー）する挙動。 |
| **Fairy Helper** | 画面上で励ましやミスへの警告を行うギャル妖精のナビゲーター。ユーザーの累計打鍵数やストリークに応じてメッセージが変化します。 |
