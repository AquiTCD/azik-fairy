# 外部辞書データインポート 概要

本機能は、AZIKタイピングゲームの「まとめ練習」および「実戦練習」で使用される語彙データを、外部のオープンソース日本語辞書データから大規模にインポートし、自動的にAZIKレベルへ振り分けた上で、クライアントに配信するシステムです。

## 目的

1. **語彙数の大幅な拡張**: 初期のAzikTデータ（約1万語）に加え、数十万〜数百万語規模の語彙を自動的にゲームに取り込み、タイピング練習のバリエーションを劇的に増やします。
2. **パフォーマンスの両立**: 語彙データが100万語規模に達するとJSONサイズが数十MBになります。これをそのままブラウザにロードするとプチフリやロード遅延の原因になるため、**「シャッフル＆3000語ずつのチャンク分割」** と **「ゲーム開始時のランダムな動的インポート」** を組み合わせて、軽量（1ファイルあたり200KB前後）かつ多様なワードプールを実現します。
3. **ライセンスの整合性 (GPL対応)**: GPLライセンスの辞書データ（SKK等）をリポジトリに含めて公開するため、プロジェクト全体をGPL-3.0ライセンスで公開し、法的なクリーンさを担保します。

## 主要な用語定義

- **サマリーステージ**: `lev1-summary`, `lev2b-summary` などの、各レベルの全ルールを統合したまとめ練習ステージ。
- **実戦ステージ**: `practice-words-1` などの、日常単語をフル拡張で練習するステージ。
- **チャンク (Chunk)**: 巨大な語彙JSONファイルを、3,000語ごとに小分けにしたJSONファイル（`[stageId]-[index].json`）。
- **マニフェスト (manifest.json)**: 各サマリー/実戦ステージが何個のチャンクに分割されているかを記録したメタデータ。

## ターゲット・ユースケース

1. **プレイヤーのゲームプレイ時**: まとめステージや実戦ステージを選択してプレイを開始すると、ブラウザが自動的にランダムなチャンクを1つだけ動的ロードし、毎回異なる単語プールで練習を行います。
2. **開発者による語彙メンテナンス時**: 開発者が新しい語彙リスト（TSV）や、辞書データをスクリプト経由で追加インポートし、チャンクを再生成・ビルドしてデプロイします。
