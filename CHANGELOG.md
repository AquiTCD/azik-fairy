# Changelog

## [v1.2.0] - 2026-06-16

- リザルト画面の妖精のセリフデータを JSON に切り出し共通化（URL クエリパラメータを大幅に短縮）
- 実戦およびお題ステージで AZIK 以外の通常ローマ字入力を許容（キー入力強制モード無効化）
- 通常のローマ字入力と比べて AZIK 入力により省力化できたキー数（SAVED KEYS）の算出とリザルト画面への表示機能を追加
- タイピング中（プレイ画面）の全スコア統計（経過時間、入力文字数、WPM、正確性、ミス数、AZIK率、省力キー数）のリアルタイム表示機能を追加

## [v1.1.1] - 2026-06-16

- 「ゅ」入力に関する問題を修正
- X（Twitter）でシェアした際に OGP 画像が表示されない問題を修正

## [v1.1.0] - 2026-06-16

- 入力不可能な問題の修正
- 「外来語拡張 [TGI/DCI/TGU]」レッスンを追加
- ゲーム画面に音声 ON/OFF ボタンを追加
- いくつかの画面でキーボード操作に対応
- 実践レッスンから AZIK ショートカット不要の単語を除外
- その他軽微なUIの修正

## [v1.0.0] - 2026-06-14

- 初回リリース
- AZIK 辞書エンジン、各種レッスン
- カスタムキー設定、厳格モード、音声フィードバック
